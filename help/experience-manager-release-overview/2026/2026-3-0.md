# 2026.03.0 AEM版本總覽影片

>[!VIDEO](https://video.tv.adobe.com/v/3483071/?captions=chi_hant&learn=on&enablevpops)

此版本中的功能著重於Experience Manager Sites、Forms、Assets和Cloud Foundation創新，並包括下列專案：

* 網站
   * Experience Production Agent更新與啟動支援

* 表單
   * IC編輯器的工作流程支援
   * 核心元件的主題編輯器支援
   * Forms的版本設定支援

* 資產
   * AEM內容顧問 — GA與Adobe應用程式
   * Microsoft Office聯結器(PowerPoint、Word、Excel)

* Cloud Foundation
   * 治理代理程式：品牌管理
   * 觸控式UI上的稽核許可權
   * OIDC登出功能
   * 暫停維護更新

* Cloud Manager
   * Cloud Manager MCP伺服器
