# 2025.2.0 AEM版本總覽影片

此版本中的功能著重於透過智慧、加速啟動和Cloud Foundation來產生內容，並包含下列專案：

## AEM Sites

* 內容片段自動標籤
* 內容片段中的遠端資產支援  編輯者

## AEM Sites、Assets、Forms

* 翻譯管理API

## Cloud Foundation

* AEM 6.5 LTS （長期支援）
* 將SLA報表延伸至Edge Delivery Services 99.99
* 按一下即可建立Edge Delivery Services網站
