# 2024.7.0 AEM版本總覽影片

此版本中的功能著重於Experience Manager Assets和Cloud Service創新，包括下列專案：

**AEM Assets**

* AEM Assets中的Content Credentials&#x200B;
* AI支援的Dynamic Media視訊標題&#x200B;
* 上傳的資產選擇器增強功能&#x200B;

**Cloud Manager**

* 客戶管理的CDN憑證的自助設定&#x200B;。
* CDN清除&#x200B;
* 流量篩選規則警報&#x200B;。
