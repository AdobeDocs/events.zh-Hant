# 2024.7.0 AEM版本總覽影片

此發行版本中的功能著重於Experience Manager Assets和Cloud Service創新，包括下列內容：

**AEM Assets**

* AEM Assets中的Content credentials&#x200B;
* Dynamic Media視訊的AI支援字幕&#x200B;
* 上傳的資產選擇器增強功能&#x200B;

**Cloud Manager**

* 客戶管理的CDN憑證的自助設定&#x200B;。
* CDN清除&#x200B;
* 流量篩選規則警報&#x200B;。
