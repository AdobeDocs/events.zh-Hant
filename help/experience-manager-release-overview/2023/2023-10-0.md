# AEM as a Cloud Service 2023.10.0 版本更新

此發行版本中的功能著重於Experience Manager Assets和Cloud Service創新，包括下列內容：

>[!VIDEO](https://video.tv.adobe.com/v/3425186/?learn=on)

**預先發佈**

>[!TIP]
>
>[了解更多關於預先發佈以及啟用方法。](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/release-notes/prerelease.html?lang=zh-Hant)

* Forms
   * （發行前） AEM Forms的新範本和主題

**版本功能**

* 資產
   * AEM Assets中的Adobe Express編輯器( *live 2023年11月16日* )

   * 大量匯入oneDrive的聯結器

* 表單
   * AEM Forms中的核心元件自訂屬性支援
