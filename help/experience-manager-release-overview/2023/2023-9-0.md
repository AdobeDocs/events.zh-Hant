# AEM as a Cloud Service 2023.9.0 版本更新

此發行版本中的功能著重於Experience Manager Assets和Cloud Service創新，包括下列內容：

**版本功能**

* 資產
   * 適用於 Adobe Express 的 AEM Assets 附加元件
   * 透過Asset Selector SDK &quot;Integrate Anywhere&quot;

* Cloud Foundation
   * 內容轉移工具3.0
   * 取消Cloud Manager中的管道
