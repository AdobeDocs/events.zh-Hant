# AEM as a Cloud Service 2023.9.0版本更新

此版本中的功能著重於Experience Manager Assets和Cloud Service創新，包括下列專案：

**發行功能**

* 資產
   * 適用於Adobe Express的AEM Assets附加元件
   * 透過資產選擇器SDK &quot;Integrate Anywhere&quot;

* Cloud Foundation
   * 內容轉移工具3.0
   * 在Cloud Manager中取消管道
