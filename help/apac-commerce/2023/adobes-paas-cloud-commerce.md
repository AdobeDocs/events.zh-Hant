# 使用Adobe的PaaS Cloud Commerce簡化您的業務

我們有一位客戶是印尼領先的運動服裝經銷商Panatrade，向您介紹他們從內部部署到Adobe Commerce Cloud的歷程，以及他們如何為電子商務業務轉向完全最佳化及受管理的解決方案而獲得回報。
