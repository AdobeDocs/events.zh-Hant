# 在 MBI 中管理資料集

直接從 Adobe Commerce 產品團隊了解 MBI Data Warehouse Manager 的一些強大功能。超越基本報告建置，了解如何利用資料事半功倍。

在本錄制的網路研討會中，您將學習如何通過加入Adobe Commerce、Google電子商務和GoogleAdWords的資料來計算廣告活動的投資回報。
