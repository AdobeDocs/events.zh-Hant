# 在 MBI 中管理資料集

直接從 Adobe Commerce 產品團隊了解 MBI Data Warehouse Manager 的一些強大功能。超越基本報告建置，了解如何利用資料事半功倍。

在這場錄製的網路研討會中，您將瞭解如何透過聯結Adobe Commerce、Google E-commerce和Google AdWords的資料，來計算廣告促銷活動的投資報酬率。
