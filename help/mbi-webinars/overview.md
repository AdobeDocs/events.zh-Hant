# MBI網路研討會

MBI網路研討會系列是MBI客戶的定期活動。 網路研討會涵蓋各種主題和最佳實踐，以便最大限度地利用MBI。

來自所有地區的與會者都可以加入現場活動，以便更好地瞭解產品並直接回答他們有關MBI功能的問題。

偶爾會重複這些主題，以適應客戶可能提出的任何新問題。
