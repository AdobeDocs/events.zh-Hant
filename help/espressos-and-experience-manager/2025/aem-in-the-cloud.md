# AEM在雲端上為何更強大

在本次研討會中，Adobe自己的業務顧問Casey Witter和技術顧問Rajan Kumar深入探討轉換至雲端的要領。 轉換至雲端可提供更強大的擴充能力和協助工具，讓企業能夠快速適應不斷變化的需求。 Casey會帶我們瞭解Adobe的可靠方法，協助您有效率地完成轉換，從徹底的規劃以及內容和後端評估開始。 此程式包含詳細的移轉計畫，能有效地配置資源。 她也會探索實施作業的關鍵工作流程、簡化程式的實用工具，以及深入瞭解預期時間表。 接著Rajan示範如何運用Adobe的Best Practice Analyzer和Cloud Acceleration Manager工具，評估您雲端的整備程度。 我們會一如既往地即時回答您的問題，解答時間一律如此。

>[!VIDEO](https://video.tv.adobe.com/v/3443023/?learn=on&enablevpops)

## 重要技巧

* **效能提升**&#x200B;由於架構、資產處理及全球資料中心更完善，Cloud Service客戶在頁面載入時間上提升了400%。

* **存取最新創新**&#x200B;雲端服務可讓您存取最新內容創新，包括快速建立網站、Adobe Jet和Studio、增強式安全性，以及符合HIPAA規範。

* **上市時間更短**&#x200B;由於CI/CD自動化、彈性部署及改善開發環境，雲端服務的客戶上市時間是原來的兩倍。

* **移轉工具和方法** Adobe提供Best Practices Analyzer (BPA)和Cloud Acceleration Manager (CAM)等工具，以評估並加速移轉程式。 建議的方法是分階段的方法**整備、規劃和設定、實施、上線和最佳化。

* **營運穩定性與支援** Adobe可確保雲端服務的高穩定性，具備頻繁備份、復原程式及全球資料中心等功能。 雲端服務也包含自動化部署流程和增強的安全措施。
