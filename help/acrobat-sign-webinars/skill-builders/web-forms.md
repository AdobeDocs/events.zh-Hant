# Acrobat Sign中的網路表單快速入門

>[!VIDEO](https://video.tv.adobe.com/v/3491645/?learn=on&enablevpops)

仍然以電子郵件形式來回傳送表格，或一次只追蹤一個合約的簽章？ Acrobat Sign中的網路表單可讓您發佈單一可簽署的表單，任何人都可以從您的網站、共用連結或內嵌頁面存取、填寫和簽署，而不需您傳送任何內容。 觀看此實作網路研討會，我們的專家會涵蓋：

* 什麼是Web Form以及它們在哪裡耀眼 — 從發佈新聘人員上線封包以供數百名員工完成的人力資源團隊，到在網站上張貼面向客戶的應用程式以大規模收集簽名的財務團隊，等等！
* 如何建立和啟動網頁表單 — 觀看如何建立第一個網頁表單的即時逐步解說，包括：上傳檔案或範本、放置表單欄位、設定簽署者驗證，以及將表單發佈為可共用的URL或內嵌程式碼。
* 如何管理和追蹤您的網路表單結果 — 即時監控提交、下載表單欄位資料、編輯或停用即時表單，並隨著使用量增加而保持您的網路表單井然有序。

無論您是要收集十人或一萬人的簽名，此工作階段都能讓您放心建立、發佈和管理您的第一個Web表單。
