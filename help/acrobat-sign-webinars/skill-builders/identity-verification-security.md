# 大規模建立數位信任：使用Acrobat Sign進行身分驗證和安全性

掌握Adobe的最新創新技術，這些創新技術著重於Acrobat Sign中的身分驗證、信任和合規性。 觀看最近的Acrobat Sign發行版本，瞭解如何透過增強的安全性控制項和現代身分識別功能，加強對每份合約的信心。 包括：

* 信任的數位簽名：由信任服務提供者所支援的憑證式數位簽名，其範圍廣泛，能確保完整性、不可否認性和長期信任。
* 全球數位身分：數位身分閘道提供全球各地高度信賴的身分驗證，包括Open Banking和政府發行的eID方案。
* 高保證、符合法規的簽署：符合業界標準與新法規（包括eIDAS 2.0）的進階與合格電子簽章。
