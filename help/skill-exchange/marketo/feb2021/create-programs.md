# 在Adobe[!DNL Marketo Engage]中建立程式

此課程會教導您所有關於方案結構的知識，包括 — 方案型別、頻道和狀態和標籤。 您也會學習到基礎計畫概念以及組成計畫的內容。

在結束本課程時，您將瞭解如何以最有效率的方式設定您的[!DNL Marketo Engage]執行個體以符合您的業務需求，以及如何建立程式以執行電子郵件和網路研討會。
