# 您的Marketo藍圖

Marketo可自訂且可擴充，因此有很多商機。 但與此同時，不良規劃或缺乏規劃，可能會使您面臨規模不良的Marketo執行個體和大量技術債。

將您的Marketo執行個體視為產品會有所幫助。 您有客戶，並且需要將執行個體開發成可讓客戶滿意的產品。 但是，直接急著進行生命週期行銷或ABM可能不會很順利。 因此您也有產品藍圖，這是您的工作方案，指導您如何實現目標。
