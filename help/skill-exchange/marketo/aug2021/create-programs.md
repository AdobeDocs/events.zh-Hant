# 在中建立方案 [!DNL Marketo Engage]

這個課程會教導您 *程式結構* 包括：方案型別、管道和狀態以及標籤。 您也會瞭解 *基礎計畫概念* 以及組成計畫的因素。

在結束本課程時，您將瞭解如何設定您的 [!DNL Marketo Engage] 執行個體以最有效率的方式滿足您的業務需求，以及如何建立程式以執行電子郵件和網路研討會。

在本次研討會結束時，現場問答將回答您所有未完成的問題。
