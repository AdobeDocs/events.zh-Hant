# 技能交流活動2023年8月 — Grow Track — 分析反轉換因素以改善客戶體驗

在本次研討會中，我們將重點討論

* 如何使用錯誤率來推動深入分析
* 找出客戶混淆的區域，以改善客戶歷程
* 在組織中實施反轉換的秘訣
