# 自動化感知 — 使用警報和註解來防止意外發生

想要立即發現問題而不用盯著圖表看嗎？ 想要在分析較舊資料時考慮異常，而不記住所發生的事情？ 使用警報和註解來讓Adobe Analytics提醒您並隨時瞭解效能的重大變更。

在本次研討會中，您將瞭解如何：

1. 立即自動使用警示來瞭解效能變更
2. 使用註解來顯示重要事件發生的時間
3. 自訂這兩者，讓只有受影響的群組能知道
