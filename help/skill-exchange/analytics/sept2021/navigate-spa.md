# 導航SPATest構建和分析中的挑戰

單頁應用(SPASingle Page App)站點設計正在興起，雖然它比多頁應用(MPA)提供了許多優勢，但它可能會對您的test構建和分析造成嚴重破壞，而不會關注正確的詳細資訊。 在本節中，我們將與您分享從Panera麵包測試和優化計畫中學到的一些最佳做法和經驗教訓。 我們將介紹資料層注意事項、test在頁面載入或視圖上交付的影響等。
