# 讓您的AEM實作發揮最大效用：有效執行網站的3個關鍵因素

在這場會議中，您將瞭解：

* 不同的網站管理模式及其優點
* 如何實作適合您組織的治理模型
* 如何隨著組織的成熟而不斷演化
