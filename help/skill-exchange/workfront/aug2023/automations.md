# 用途驅動自動化

運用自動化，可以讓使用者專注於需要獨特技能與專業知識的工作，而非阻礙他們工作的例行性和重複性的工作，藉此釋放巨大的效率潛力。

在這場會議中，來自Zimmerman Advertising的Jennafer Higgs、Jonathan Cabre和Malcom Benites將分享經證實可建立更有效工作流程的實際自動化範例。

他們將會同時分享Fusion和非Fusion自動化，包括：

·如何自動關閉專案
·產生子範本
·合併已核准的資產檢閱
