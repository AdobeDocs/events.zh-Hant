# 詢問專家 — 文字模式報告簡介

文字模式可透過使用標準模式介面中不可用的欄位和計算，擴展您的報告功能，以建立更複雜的檢視、篩選器、分組和提示。 Workfront首席技術顧問Tony Messam將討論API Explorer、基本文字模式元件，並分享即時示範。 此網路研討會錄製於2020年7月15日。
