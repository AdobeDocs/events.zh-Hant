# 前3項 — 如何開始使用里程碑

讓我們討論里程碑路徑！ 這是您的「隨選Workfront CSM」 Cynthia Boon，我想與大家分享我對里程碑報表的熱愛，以及入門的簡單程度。

對於尋找專案狀態，但可能不需要檢視專案完整詳細資訊的更高層級領導甚至請求者來說，這份報表是絕佳選擇。

在您的專案控制面板上加入「里程碑報告」或共用此報告（更好，自動排程定期傳送）是讓相關各方(甚至非Workfront使用者)持續參與專案進度的好方法。
