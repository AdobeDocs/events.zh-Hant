# Nathan的5個Faves — 適用於Workfront世界的基本API

好奇Workfront API到底能為您做些什麼？ 加入Adobe技術支援工程師III Nathan Johnson的行列，他將剖析基礎知識，並將API帶入真實世界。

在這場會議中，Nathan會先逐步解說Workfront API的基本概念、內容、運作方式以及重要原因，然後再深入探討他最愛的五個使用案例，以展示其運作中的力量。
