# 使用Workfront Fusion自動化命名慣例的5個秘訣

讓Workfront環境維持井然有序，一開始會採用一致的命名慣例，但手動強制執行這些慣例可能既費時又容易出錯。 在這場與Adobe Professional Services和現場工程合作舉辦的網路研討會中，瞭解如何運用Workfront Fusion自動將命名標準套用至專案、任務、問題等。

我們將透過實際案例介紹Fusion的使用方法：

- 根據表單輸入或中繼資料動態產生專案名稱
- 標準化任務和問題名稱，以方便報告和顯示
- 利用遞增的數字來標示物件，並確保執行個體的唯一性

無論您是剛開始使用Fusion，還是想要擴充您的自動化，您都可以從實用的範例和靈感中解脫出來，以簡化您的Workfront設定。
