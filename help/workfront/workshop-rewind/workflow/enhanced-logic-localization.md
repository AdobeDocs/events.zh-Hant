# 擷取更聰明、可擴充至全球 — 自訂Forms與Workfront本地化

在進階企業營運(AEO)功能事件系列的第二次會議上，我們的Adobe產品經理Khachatur Sedrakyan和Kristine Tumanyan探索了四種新的進階邏輯型別、顯示、可編輯性、條件式格式和預設值，以實現更動態、自動化和資料精確的攝取體驗。

此外，他們引進了新的本地化增強功能，可讓團隊為使用者輸入的標籤新增自訂翻譯，協助全球組織建立更一致、可存取且符合區域協調的Workfront體驗。
