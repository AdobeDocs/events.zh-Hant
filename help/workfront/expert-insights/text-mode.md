# Workfront專家深入分析：文字模式與Katherine Lanning

歡迎使用最新的Expert Insights！  我是Cynthia Boon，您的「隨選」Workfront CSM，而本集我們有Workfront文字模式Jedi、Katherine Lanning、來自IQVIQ的Workfront解決方案工程師！  在這一集裡，Katherine分享她的故事、她對文字模式報告的所有事情的熱情，並示範她如何建立她的「健全管理」儀表板！

文字模式可能很具挑戰性，但回報卻驚人。  記住Katherine的建議！

* 「如果能看見，就能變更。」  — 在標準報表上翻轉成文字模式以檢視如何建置欄。
* 修修補補以學習。  — 新增欄位、切換到文字模式並進行實驗，以瞭解驅動資料的原因。
* 使用純語言。  — 標題欄當作問題或清除陳述式，讓非管理員可以輕鬆解讀。
* 新增說明。  — 以文字模式自訂暫留文字，以說明資料並引導使用者。
* 讓視覺效果易於存取。  — 不要僅依賴顏色（例如紅色/綠色）。 使用不同的圖示或標籤。
* 善用社群。  — 仰賴Experience League、Cookbook、API Explorer和其他使用者。 不要再創造轉輪。
* 會持續保留，但會中斷。  — 如果卡住，請合上筆記型電腦，散步，然後帶著清新的眼睛回來。

## 關於喇叭

Katherine是IQVIA Digital的Workfront解決方案工程師和Fusion開發人員。 當被問到她的職業時，她會回答「我是某人」，比如「某個人真的應該會想出更好的辦法去做。」 她的背景涵蓋企業財務、IT以及成人飲料、行銷與廣告甚至私人保安等產業的系統管理。

她住在俄亥俄州西南部，與丈夫偶遇、他們的狗「Bettis」、兩輛摩托車和足夠三個人的愛好。 理想的週末包括黑刺會議、探索荒山、使用Xbox享受高品質時間，或烘焙新的芝士蛋糕食譜。

(是的，所有這些都完全正確。 她不是那個玩破冰機「兩個真話和一個謊言」遊戲的人。)
