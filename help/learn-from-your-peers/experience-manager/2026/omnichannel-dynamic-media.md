# 隨時隨地提供一次體驗：使用Adobe Dynamic Media擴展全通路體驗

您是否想知道如何簡化在AEM Assets中影像和視訊的轉換和最佳化？

加入Adobe Experience Manager Champion Vengadesh Shanmugavelu，深入瞭解如何在Adobe Experience Manager Assets中使用Dynamic Media。 他將介紹擴展數位內容製作的關鍵元素，包括：

>[!VIDEO](https://video.tv.adobe.com/v/3481556/?captions=chi_hant&learn=on&enablevpops)

* Adobe Experience Manager Dynamic Media如何使用設定檔設定、智慧型轉譯和即時最佳化，來啟用智慧型媒體發佈。
* 如何透過AEM核心元件啟用Dynamic Media，並使用Dynamic Media Open API在整個企業內擴充其功能。

無論您是行銷人員、架構師或開發人員，此網路研討會都會向您展示如何只傳送一次，並透過效能、一致性和控制力在各個地方擴展。

Vengadesh Shanmugavelu是Publicis Sapient高級工程經理。 他在內容管理解決方案架構、設計、開發、實作、訓練及支援AEM和Interwoven產品套裝方面擁有15年以上的經驗，並在AEM和Adobe Marketing Cloud解決方案方面擁有7年以上的經驗。 彼為Adobe認證專家（擔任AEM架構師及首席開發人員）、Adobe認證專家（擔任Adobe Target架構師）及Adobe合格體驗平台工程師。 他自2022年起擔任AEM冠軍。
