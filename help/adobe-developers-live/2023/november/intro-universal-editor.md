# AEM通用編輯器簡介

通用編輯器是新一代AEM內文中頁面編輯器，可解決AEM頁面編輯器或SPA編輯器目前的限制。 它獨立於架構、演算引擎和資料來源，因此提供開發人員所追求的所有自由，並真正實現通用。
