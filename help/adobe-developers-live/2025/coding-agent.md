# AI程式碼大戰：好、壞和醜

探索AI編碼代理程式在真實世界Edge Delivery Services專案中的執行方式。 瞭解他們的優勢、挑戰，以及對開發人員體驗的影響。 這場來自Adobe Developers Live 2025的會議展示了實用的自動化與「vibe coding」的對比，突顯了AI的優秀之處和掙扎之處。 加入Lars Trieloff的行列，測試代理程式工程，比較領先的AI編碼代理程式，以揭示數位體驗未來的洞見。

>[!VIDEO](https://video.tv.adobe.com/v/3477659/?captions=chi_hant&learn=on&enablevpops)

這個會議 — Coding Agent Showdown： The Good， the Bad， and the Ughtly — 將代理工程付諸測試。 Lars Trieloff比較真實Edge Delivery Services專案上領先的AI程式碼代理程式，揭示開發人員體驗和代理程式體驗交匯的地方。 觀看AI在何處閃耀、在何處掙扎，以及實際自動化與「視覺編碼」之間有何差異。 從聖荷西現場錄製。

特別感謝我們的贊助商Algolia和Ensemble對Adobe Developers Live 2025的支援。
