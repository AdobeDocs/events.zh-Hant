# 使用Edge Delivery Services (EDS)現代化Qantas.com

瞭解澳航如何使用Adobe Experience Manager Edge Delivery Services改版網站。 此轉換將發佈時間從數週縮短至數分鐘，簡化開發工作，並透過更簡潔的語意標籤改善SEO。 瞭解從內部部署到邊緣架構的轉變，以及如何利用JavaScript工作流程擴展人才庫。 深入瞭解此現代方法如何提升撰寫速度，並支援公司治理目標。 加入對話並探索即將舉辦的Adobe活動，以獲得更深入的見解。

>[!VIDEO](https://video.tv.adobe.com/v/3477681/?learn=on&enablevpops)

這場Fireside聊天 — 透過Edge Delivery Services現代化Qantas.com — 特別推薦了Accenture Song的解決方案架構師Tim Lavelle和Adobe的資深產品專員Glen Lawson，他們在雪梨現場錄製。 他們討論了澳航的主要重建專案和轉移到Adobe Experience Manager Edge Delivery Services，重點說明從內部部署到邊緣架構如何將發佈時間從幾週減少到幾分鐘。 瞭解這種現代方法如何簡化開發、使用JavaScript式工作流程擴展人才庫，以及透過更簡潔的語意標籤和通用編輯器改善編寫速度和SEO。

特別感謝我們的贊助商Algolia和Ensemble對Adobe Developers Live 2025的支援。
