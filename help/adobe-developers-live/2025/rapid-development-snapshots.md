# 凍結、還原、創新：快速開發環境的動手操作快照，適用於雲端開發

探索Adobe Experience Manager Cloud Service中的快速開發環境(RDE)如何實現雲端原生的Swift互動。 瞭解快照功能如何讓開發人員得以凍結、還原及重複使用完整的RDE狀態。 這可讓您使用Adobe I/O CLI RDE外掛程式來簡化測試、立即復原及加快開發週期。 提升您的雲端開發效率，並符合公司治理政策。

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

本課程 — 凍結、還原、創新：快速開發環境動手拍攝雲端開發快照 — 特別說明Adobe的軟體開發工程師Remo Liechti，並預先錄製了Adobe Developers Live 2025年的影片。 Remo展示快速開發環境(RDE)如何在Adobe Experience Manager Cloud Service中啟用快速的雲端原生反複專案，以及新的快照功能如何讓開發人員凍結、還原及重複使用完整的RDE狀態。 瞭解如何使用AEM CLI RDE外掛程式在Adobe I/O Cloud中簡化測試、立即復原及加快開發週期。

特別感謝我們的贊助商Algolia和Ensemble對Adobe Developers Live 2025的支援。
