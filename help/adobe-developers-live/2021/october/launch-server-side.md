# 啟動伺服器端很好，但如果可以……

2021年4月，Evolytics在Adobe Experience Platform發佈了第一個第三方創作的事件轉發功能擴展插件（以前稱為「啟動伺服器端」）。 開發事件轉發擴展的過程與客戶端標籤屬性上的過程非常相似，但需要注意一些重要的差異。 在本課程中，Evolytics工程師將帶領您完成設計、開發、測試和發佈事件轉發擴展的過程。
