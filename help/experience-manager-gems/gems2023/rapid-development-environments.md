# 透過快速開發環境加速AEM as a Cloud Service開發

此網路研討會已在2023年2月22日舉行。 請檢視下面的錄製。

## 工作階段說明

為了部署變更，目前的雲端開發環境需要使用採用廣泛計畫碼安全性和品質規則（稱為CI/CD管道）的程式。 對於需要快速反複變更的情況，Adobe已推出快速開發環境（簡稱RDE）。

RDE可讓開發人員快速部署和檢閱變更，將測試經證實可在本機開發環境中運作的功能所需的時間減至最少。
