# 如何充份運用DAM Feature Pack{#how-to-get-the-most-out-of-your-dam-feature-pack}

適用於AEM 5.6.1的DAM FP2著重於利用創意專案功能，為AEM使用者提供管理其資產來源的獨特方式。 此課程涵蓋創意專案工作流程的逐步操作，內容包括：

* 建立/編輯/上傳快照清單
* 對照快照清單上傳資產
* 檢閱及核准上傳的資產

此課程也會示範任務表面設定、自訂專案工作流程範本、產品階層建立的能力。
